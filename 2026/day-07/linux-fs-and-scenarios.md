
### Linux File System Hierarchy 
  Linux is all about processes ,there are 19 of file system hierachies in linux  (/,/bin,/sbin,/root,/home...etc)
  
  ##Imprtant direcries-(must know)
  /root - (main file those contain  various directory & the starting point of everything , ) 
  /home - User home directories,when i list /home directory then it shows where i am who i am permissions and some kind of information by running this command ls -l   /home 
  /etc  - configuration file (using this ls -l /etc cmd , i saw number of lines,those are all configurations files)
  /var - variable files (ls -l /var )
  /log - logs file
  /tmp - Temporary files
  
  ##Additional Directories (Good to Know):
  /bin - Essential command binaries
  /usr/bin - User command binaries
  /opt - Optional/third-party applications
