1.Process management
ps aux - show all processes
top - Running processes view
htop - Enhanced processe 
kill - graceful stop
kill -9 - force kill
uptime - system load and runtime 
free -m - memory usage 
pidof nginx - get PID of service

2.File system
pwd - present working directory (Shows current directory)
ls -lah - list files (long,all,human-readable)
cd /path - change directory
mkdir -p dir/subdir -create nested directories
rm -rf dir - force delete directory (dengerous)
cp -r src dest - copy files/diretories
mv old new - move ro rename files
du -sh * -disk usage per item
df -h disk usage space
star file -deleted file info
cat - for view inside of file
wc - word count 

3.Networking troubleshooting
ping google.com - check connectivity
ip a - show ip address
dig google.com - shows DNS resolution
curl https://example.com -in simple word open website without browser
